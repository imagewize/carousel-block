## Version Number Locations

When updating the version, ensure you update all of these files:
- `plugin.php` (2 locations: header comment and VERSION constant)
- `package.json`
- `src/carousel/block.json`
- `src/slide/block.json`

Then run `npm run build` and commit all changes including the built files.
